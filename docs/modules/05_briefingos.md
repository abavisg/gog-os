# Module Spec: BriefingOS

## Purpose

Create the morning command centre.

## Build/runtime model

Claude Sonnet 4.6.

## Inputs

- Latest EmailOS report.
- Latest CalendarOS brief.
- Open TaskOS tasks.
- Optional NewsOS summary.
- Optional ReflectionOS carry-forward.

## Output

Daily Markdown/HTML brief.

## Acceptance criteria

- Top 3 priorities.
- Schedule constraints.
- Email actions.
- Risks.
- Suggested plan.
- Missing inputs clearly marked.
