# Module Spec: LearningOS

## Purpose

Support structured learning tracks.

## Model

Claude Sonnet 4.6 for planning, Haiku 4.5 for simple daily formatting.

## First tracks

- Python.
- React.
- AI agents.
- RAG.
- Rust.

## Acceptance criteria

- Daily lesson is connected to a track.
- Practice task is concrete.
- Progress is logged.
