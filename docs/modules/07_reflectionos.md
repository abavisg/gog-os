# Module Spec: ReflectionOS

## Purpose

End-of-day and weekly review.

## Model

Claude Sonnet 4.6.

## Commands

- `/end-day`
- `/weekly-review`

## Outputs

- Completed.
- Slipped.
- Decisions.
- Lessons.
- Follow-ups.
- Tomorrow seed list.

## Acceptance criteria

- Direct and useful.
- Identifies overload and drift.
- Feeds carry-forward into next morning brief.
