# Module Spec: AgentOS

## Purpose

Track commands, skills, models, permissions, and workflow dependencies.

## Model

Claude Sonnet 4.6 for implementation. Opus 4.8 for architecture review.

## MVP

- Static module registry.
- Command registry.
- Skill registry.
- `/gogos-status`.

## Acceptance criteria

- User can see what exists, what is planned, and what model each command should use.
