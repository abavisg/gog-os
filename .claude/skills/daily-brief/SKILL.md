# Daily Brief Skill

Create one practical daily brief.

## Required sections

1. Top 3 priorities.
2. Schedule and constraints.
3. Email actions.
4. Task focus.
5. News signals.
6. Health/learning/content reminders if configured.
7. Risks.
8. Suggested plan.

## Rules

- Be specific.
- Do not produce generic motivation.
- Mark missing inputs clearly.
- Prioritise trade-offs, not long lists.
