# /morning-brief

Model: Claude Sonnet 4.6

Purpose: generate the daily GogOS command centre.

Inputs:

- Latest EmailOS report if available.
- Latest CalendarOS brief if available.
- Local task list if available.
- NewsOS feed summaries if available.
- Activity/reflection carry-forward if available.

Output:

- Top 3 priorities.
- Schedule.
- Email actions.
- Calendar prep.
- News signals.
- Health/learning/content reminders when configured.
- Risks and decisions.
